# Active Directory User Management Project

## Project Overview

This project demonstrates common Active Directory administrative tasks performed by IT Support and Help Desk professionals.

The purpose of this project is to showcase user account management, password administration, account security, and troubleshooting procedures within an Active Directory environment.

---

## Skills Demonstrated

- Active Directory
- User Account Management
- Password Resets
- Account Unlocks
- Security Best Practices
- IT Documentation
- Customer Support

---

## Scenario 1: Password Reset

### Problem

A user cannot access their account after forgetting their password.

### Troubleshooting Steps

1. Verified user identity.
2. Opened Active Directory Users and Computers.
3. Located user account.
4. Reset password.
5. Required password change at next login.
6. Confirmed successful login.

### Resolution

Password was successfully reset and account access restored.

---

## Scenario 2: Locked User Account

### Problem

A user entered an incorrect password multiple times and became locked out.

### Troubleshooting Steps

1. Verified user identity.
2. Located user account.
3. Reviewed account status.
4. Unlocked account.
5. Tested login.

### Resolution

User account was unlocked and normal access restored.

---

## Scenario 3: New Employee Setup

### Problem

A new employee requires access to company systems.

### Troubleshooting Steps

1. Created new user account.
2. Assigned username.
3. Set temporary password.
4. Added user to required groups.
5. Verified account creation.

### Resolution

New employee account was successfully configured.
